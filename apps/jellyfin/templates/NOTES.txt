Jellyfin has been deployed!

{{- if .Values.ingress.enabled }}

Access Jellyfin at: http://{{ .Values.ingress.host }}

{{- else }}

To access Jellyfin, port-forward the service:
  kubectl port-forward svc/{{ include "jellyfin.fullname" . }} 8096:{{ .Values.service.port }}

Then visit: http://localhost:8096
{{- end }}

Configuration and metadata stored in:
  PVC: {{ include "jellyfin.fullname" . }}-config
  Path: /config

{{- if .Values.persistence.media.enabled }}
Media files accessible from:
  PVC: {{ include "jellyfin.fullname" . }}-media
  Path: /media

To share this storage with MeTube, install MeTube with:
  helm install metube ../metube --set persistence.existingClaim="{{ include "jellyfin.fullname" . }}-media"
{{- end }}

First-time setup:
1. Open Jellyfin in your browser
2. Complete the initial setup wizard
3. Add a media library pointing to /media
4. Start enjoying your media!
